Catch and beginning with git

Goals of this assignment
  • Start to become more familiar with command line git
  • Start to learn how to use the Catch C++ unit testing framework.
Here is a simple walk thru using the Catch C++ framework to do TDD (test driven development) on a simple / silly C++ class called IntegerInfo.

Since we'll all be working on files with a similar name, we won't be able to work in the same repository space as for the previous two homework assignments. When you start working on this assignment (after doing the 'git pull' to get the files). You will:

  • Create a branch 
    % git branch YourName
% git checkout YourName
OR
% git checkout -b YourName
  • Edit, add, commit your files and changes
  • Push your branch to the remote repository 
    % git push -u origin YourName
